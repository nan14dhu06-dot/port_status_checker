
# Port Status Checker

## Description
A simple Python project that checks whether a specific port on a target system is open or closed using socket programming.

## Features
- Checks port status
- Uses Python socket module
- Beginner-friendly cybersecurity project
- Supports IP address or website input
- Includes basic error handling

## Technologies Used
- Python
- Socket Programming

## How to Run

1. Install Python  
2. Save the file as `port_checker.py`  
3. Open terminal or command prompt  
4. Run the command:

```bash
python port_checker.py
```

## Example

Input:
```bash
Enter target IP address or website: google.com
Enter port number: 80
```

Output:
```bash
Port 80 is OPEN on google.com
```
